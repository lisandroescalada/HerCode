Creamos este ambiente donde las mujeres se sientan bienvenidas y apoyadas, donde se celebren sus habilidades y logros en tecnología. Queremos cambiar la manera en que las personas piensan sobre lo que pueden hacer las mujeres en tecnología y ayudarlas a alcanzar sus metas.

Cursos

Desarrollo de VideoJuegos

Explora desde la idea hasta la implementación con Unity o Unreal. Aprende programación de gráficos, inteligencia artificial

Robótica

Curso introductorio para programar y construir robots, explorando mecánica y sensores. Desarrolla habilidades prácticas y sigue las últimas tendencias.
Programar con Python

Domina Python, desde conceptos básicos hasta desarrollo web con Django y análisis de datos. Prepárate para diversos desafíos de programación.
Diseño de Paginas Web

Aprende desde HTML y CSS hasta frameworks como React. Diseña interfaces atractivas y crea aplicaciones web dinámicas para clientes y servidores.
Ciencia de Datos

Curso centrado en el manejo de grandes conjuntos de datos, aprendizaje automático y visualización. Utiliza Python, Pandas y Matplotlib para extraer información valiosa.
Programar con PHP

Curso de desarrollo web del lado del servidor con PHP. Construye aplicaciones dinámicas, interactúa con bases de datos y crea sitios web eficientes. Habilidades clave para sistemas web robustos.
